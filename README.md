# Smart Leads Dashboard

A full-stack Lead Management Dashboard built using MERN Stack.

## Tech Stack

### Frontend
- React
- TypeScript
- TailwindCSS

### Backend
- Node.js
- Express
- MongoDB
- JWT Authentication

## Features

- JWT Authentication
- Role Based Access
- Leads CRUD
- Filtering & Search
- Pagination
- CSV Export
- Debounced Search
- Responsive UI

## Setup

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Environment Variables

```env
MONGO_URI=
JWT_SECRET=
```

## Deployment

Frontend: Vercel  
Backend: Render
